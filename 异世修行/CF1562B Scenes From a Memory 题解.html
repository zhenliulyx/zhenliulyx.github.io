    CF1562B Scenes From a Memory 题解

    题目大意

    • 给定一个整数 \(n\)，问：这个数最多能去掉多少位，才能变成非质数（合数或等于1）。
    • 保证可以删除一些数字使给定的数变为获得非质数。

    思路

    • 易得答案非质数必为1位或两位。因为 \(k\) 很小，直接暴力求解。

    简单证明如下：

    1、如果一个数字包含数字“1”、“4”、“6”、“8”、“9”，那么该数字就是答案，直接输出。

    接下来处理不含这五个数（只含“2”、“3”、“5”、“7”）的情况。

    2、如果该数包含两个相等的数，即可删成“22”、“33”、“55”、“77”的格式。能被11整除。

    3、如果2、5不在数字开头，那么我们再次找到了位数为2的答案。（以2、5结尾的两位数必为合数。

    4、最后只剩下2、5在开头的情况。因为必定有答案，所以只有“237”，“ 273”, “537”,“ 573”的情况了，它们都可以删成2位合数。

    证毕。
